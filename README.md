                                          Flutter Alarm App
A Flutter mobile application that helps users sync with nature’s rhythm through beautiful onboarding screens, location and alarm notifications.

Features:
-3 Onboarding Screens With animated .mp4 videos, Next button, dot eclipses and a “Skip” button.
-Fetches user’s current location using permission handler and Geolocator.
-Pick time & date, list of alarms  and receive notifications when alarms go off.
-Local notifications implemented with flutter_local_notifications for  local alerts.
-Save alarms using shared_preferences.
-Fully responsive UI following the provided Figma design guidelines.

Tools &  Packages:                                                    
 
 provider                       State management                     
 geolocator                     Get user location                    
 geocoding                      Convert coordinates to address       
 permission_handler             Manage runtime permissions           
 flutter_local_notifications    Local notification for alarms        
 shared_preferences`            Local storage for alarms            
 video_player                   Play  .mp4  onboarding videos        
 intl                           Format date & time                   
 Oxygen font                    Google Fonts
